# Overview
Djinn is going to use the [Open-core](https://en.wikipedia.org/wiki/Open-core_model) business model, which is the same that is used by:
- [GitLab](https://about.gitlab.com/)
- [Redis](https://redis.io/)
- [Neo4j](https://neo4j.com/)
